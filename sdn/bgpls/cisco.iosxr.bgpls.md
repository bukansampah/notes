
Example to enable BGP-LS in IOS-XR

Config example
```
router isis DEFAULT
 distribute bgp-ls                 
!
router bgp 11
 address-family link-state link-state
 !
 neighbor 172.16.17.2
  remote-as 11
  local address 172.16.17.101
  session-open-mode both
  address-family link-state link-state
  !
 !
!
```
